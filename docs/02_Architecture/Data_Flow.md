# Data Flow

## Program Import

Excel file  
→ Parser  
→ Normalized program entities  
→ Validation report  
→ User confirmation  
→ Stored program

## Training Entry

User action  
→ Session context  
→ Exercise context  
→ Set context  
→ Timestamp  
→ Stored journal event

## Analysis

Stored observations  
→ Relevant historical comparison set  
→ Rule-based and statistical processing  
→ Evidence summary  
→ Confidence/uncertainty statement  
→ User-facing observation
