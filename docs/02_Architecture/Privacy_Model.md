# Privacy Model

## MVP Privacy Principles

- Avoid storing names when unnecessary
- Avoid exact birthdate
- Avoid genetic information
- Avoid medical claims
- Minimize identifying metadata
- Give the user control over media
- Separate raw records from analytics
- Make exports possible
- Do not use user data for unrelated purposes

## Anonymous Profile Fields

- Age or age range
- Sex, when analytically relevant and voluntarily provided
- Height
- Weight
- Body fat percentage
- Training experience
- Strength markers
- Work type and stress context

## Principle

Collect only what improves the user's own training record.
