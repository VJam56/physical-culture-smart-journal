# Guided Journal Experience

The journal is generated from the user's interactions.

## Journal Entry Types

- Structured input
- Free text
- Voice recording
- Photo
- Video
- Meal deviation
- Supplement deviation
- Recovery observation
- Session reflection

## Journal View

The historical view should reconstruct the session chronologically while preserving structural context.

Example:

- 6:12 PM — Session started
- 6:20 PM — Bench set 1 completed
- 6:21 PM — Voice note attached to bench set 1
- 6:28 PM — Bench set 2 completed
- 6:29 PM — Camera media attached to bench set 2
