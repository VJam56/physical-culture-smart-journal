# Training Session Interface

## Primary Screen

The session screen should show:

- Exercise name
- Planned sets, reps, load, RPE/RIR
- Actual entry fields
- Completion control
- Text note control
- Microphone control
- Camera control

## Set-Level Controls

The microphone and camera buttons must be visually and structurally associated with the exact set row.

Every captured item automatically receives:

- Session ID
- Exercise prescription ID
- Performed set ID
- Timestamp
- Sequence number

## UX Goal

The athlete should be able to record an observation immediately without navigating away from the set.
