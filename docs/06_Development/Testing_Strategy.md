# Testing Strategy

## Unit Tests

- Excel parsing
- Data normalization
- Session calculations
- Sequence ordering
- Baseline calculations

## Integration Tests

- Program import to database
- Session creation to journal storage
- Media metadata linkage
- History reconstruction

## User Tests

- Can today's session be opened quickly?
- Can a set be recorded with one hand?
- Can a voice or camera entry be attached without losing context?
- Can the user recover from accidental navigation?
- Does the workflow reduce labor compared with the current process?
