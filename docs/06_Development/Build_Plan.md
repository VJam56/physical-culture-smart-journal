# MVP Build Plan

## Phase 1 — Repository and Documentation

- Establish source of truth
- Lock MVP
- Define naming and decision process

## Phase 2 — Data Model

- Finalize Excel-derived entities
- Finalize database schema
- Create migrations
- Seed sample data

## Phase 3 — Program Import

- Parse current Excel program
- Validate structure
- Produce import report
- Store normalized program

## Phase 4 — Training Session Interface

- Display session
- Record performed sets
- Add set notes
- Add microphone and camera hooks
- Autosave

## Phase 5 — Recovery and Baselines

- Recovery inputs
- Meal baseline
- Supplement baseline
- Deviation logging

## Phase 6 — History

- Session reconstruction
- Exercise history
- Media and note review

## Phase 7 — Observation Engine

- Rule-based comparisons
- Baseline calculation
- Evidence summaries
- Uncertainty statements

## Phase 8 — Personal Trial

- Use during real training
- Log friction points
- Fix defects
- Reject premature scope expansion
