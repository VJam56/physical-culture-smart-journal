# User Stories

- As an athlete, I want today's workout ready when I open the app so I do not prepare it manually.
- As an athlete, I want to attach a note to the exact set so context is never lost.
- As an athlete, I want a microphone button beside the set so I can record observations without typing.
- As an athlete, I want a camera button beside the set so I can capture technique or documentation immediately.
- As an athlete, I want my standard meals and supplements preloaded so I only record deviations.
- As an athlete, I want to review past sessions as a coach would.
- As a coach, I want the athlete's history organized chronologically and structurally.
- As a privacy-conscious user, I want the system to avoid unnecessary identifying information.
