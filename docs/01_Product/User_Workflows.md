# User Workflows

## Primary Training Workflow

1. Open app
2. View today's planned session
3. Confirm or override session
4. Review exercise prescription
5. Begin exercise
6. Record each set
7. Add text, voice, photo, or video immediately after the set
8. Complete session
9. Add session-level reflection
10. Save automatically

## Pre-Session Workflow

- Confirm sleep
- Confirm stress
- Confirm energy/readiness
- Confirm meal baseline or deviations
- Confirm supplement baseline or deviations

## Historical Review Workflow

1. Open history
2. Filter by date, exercise, block, or program
3. Review planned versus actual performance
4. Review linked notes and media
5. View trend summaries
6. Form human judgment

## Coach Workflow

The coach receives a coherent record rather than fragmented recollection:

- Program context
- Actual performance
- Immediate athlete observations
- Recovery context
- Historical comparison
