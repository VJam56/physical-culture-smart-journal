# MVP Scope

**Version:** 1.0.0  
**Status:** Locked

## Must Do

- Import the user's current training program from Excel
- Present the correct training session
- Allow user override when necessary
- Record planned and actual training performance
- Capture set-specific written notes
- Capture set-specific voice notes
- Capture set-specific camera media
- Record simple recovery inputs
- Record planned meals and deviations
- Record supplement baseline and deviations
- Preserve chronological entry order
- Display historical session records
- Produce cautious, evidence-based observations

## Deferred

- Broad wearable ecosystem integrations
- Genetic data
- Bloodwork integration
- Medical interpretation
- Epidemiological use
- Social platform features
- Coaching marketplace
- Public leaderboards
- Automated training prescriptions
- Additional smart gadgets beyond the initial baseline

## Scope Rule

No new feature enters the MVP unless it directly improves:

1. Session preparation
2. Training execution
3. Immediate observation capture
4. Historical review
