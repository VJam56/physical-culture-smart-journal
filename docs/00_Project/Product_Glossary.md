# Product Glossary

## Guided Journal

A journal where structured interactions—sliders, selections, notes, media, and completed sets—become journal entries.

## Observation Engine

A system that organizes relationships and presents evidence without making medical or absolute claims.

## Session

A scheduled or manually selected training event.

## Set Context

The exact exercise, set number, time, and planned prescription associated with an entry.

## Historical Coherence

The value created when repeated observations are preserved in a consistent and comparable structure.

## Baseline

A repeatable reference state such as typical meals, supplement stack, body metrics, or usual performance.

## Readiness

A contextual summary of current recovery and performance-related inputs. It is not a diagnosis.
