# Open Questions

## Database and Identity

- Will the personal MVP support only one local user initially?
- Should user authentication be deferred until multi-device use is required?
- Which database will be selected after local development begins?

## Program Import

- Which workbook sheets are authoritative?
- Are merged cells or formulas used?
- Are loads stored as values, percentages, or formulas?
- How are weeks, days, and exercise blocks represented?
- Which notes must be preserved exactly?

## Session Behavior

- Can a user add an unplanned exercise?
- Can a performed set exist without a prescription?
- How should warm-up sets be represented?
- How should supersets and circuits be represented?

## Media

- Are photos and videos stored locally first?
- Should media capture be optional per set?
- What retention and export behavior should apply?
