# Project Status Addendum — v1.3.2

**Status:** Accepted  
**Phase:** Implementation  
**Active Milestone:** v1.4.0 — Excel Intelligence

## Newly Accepted Governing Direction

- Protect the athlete's concentration.
- Treat cognitive and psychological bandwidth as a design resource.
- Use the Meet Platform experience as the primary session-screen metaphor.
- Apply the rule: every interaction must reduce burden or preserve coaching-quality evidence.
- Apply the rule: capture first, organize later.

## Scope Effect

This update does not add a new MVP feature.

It clarifies how existing features must behave:

- Program display prepares the objective.
- Set logging minimizes decisions.
- Microphone and camera preserve immediate evidence.
- Autosave removes administrative burden.
- Analysis remains outside the critical lifting workflow.
