# ADR-004: Voice and Camera Media Are Linked to Exact Sets

**Status:** Accepted  
**Date:** 2026-07-15

## Decision

Microphone and camera controls belong beside the set and inherit the set's full context.

## Reason

Immediate capture reduces effort and preserves the user's evaluation at the moment it occurs.

## Consequences

Media metadata requires session, exercise, set, timestamp, and sequence linkage.
