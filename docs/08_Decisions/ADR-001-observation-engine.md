# ADR-001: The Application Is an Observation Engine

**Status:** Accepted  
**Date:** 2026-07-15

## Decision

The application will organize evidence and present observations. It will not diagnose the user or claim unjustified causation.

## Consequences

- AI output must distinguish fact from inference.
- Uncertainty must be stated.
- Source data must remain reviewable.
- Medical functionality is excluded from the MVP.
